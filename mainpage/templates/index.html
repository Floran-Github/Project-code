Floran Classroom

Floran Classroom is the platform where you can manage all your institiution staff work much more easier than your usual method.
Here all the student and staff registeration are done by the database that admin user will create.
Floran Classroom is capable for generationg the marksheet and attendance report of one division just by selecting the year and divison of class whose report is to be generate

Features

  • Record daily attendance
  • Publishing assignment by teacher
  • Submission of assignment by student
  • Grading
  • Show student how much marks they obtained in each subject in graphical method and also show them which subnissions are pending
  • Show Teacher graphical representation of student daily attendance
  • Generating report
  • Allow admin user to upload the csv data to database
  • Generate user of student and teacher automatically from database
